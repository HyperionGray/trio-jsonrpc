Trio JSON-RPC
=============

This project provides an implementation of `JSON-RPC v
2.0 <https://www.jsonrpc.org/specification>`_ based on
`sansio-jsonrpc <https://github.com/hyperiongray/sansio-jsonrpc>`_ with all of the I/O
implemented using the `Trio asynchronous framework <https://trio.readthedocs.io>`_.
